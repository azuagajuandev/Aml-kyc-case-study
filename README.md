# 💰 AML/KYC Case Study

> Anti-Money Laundering (AML) and Know Your Customer (KYC) 
> analysis combining audit expertise with data analytics techniques.

---

## 🎯 Project Goal

Apply data analytics to detect suspicious transactions, identify 
high-risk customer profiles, and demonstrate AML/KYC compliance 
workflows — bridging financial regulation knowledge with 
modern data tools.

---

## 🛠️ Tools & Skills

- **Python / Pandas** — Transaction analysis
- **SQL** — Customer data queries
- **Power BI / Tableau** — Risk dashboards
- **Excel** — Compliance workflows

---

## 📊 What's Included

- ✅ Customer risk scoring model
- ✅ Transaction monitoring analysis
- ✅ Suspicious pattern detection
- ✅ PEP (Politically Exposed Persons) flagging logic
- ✅ Geographic risk analysis
- ✅ AML dashboard mockup
- ✅ Case study documentation
